### node update방법

```
node -v  # 현재 버전 확인
npm cache clean -f # npm 캐시 제거
npm install -g n # node버전을 관리 하는 n모듈 설치

# n stable : 안정버전 
# n latest : 최신버전
# n lts : lts버전
# n x.x.x : 특정 버전
```
